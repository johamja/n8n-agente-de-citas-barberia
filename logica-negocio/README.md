# 💼 Lógica de Negocio - Barbería

Esta carpeta contiene toda la información, configuración y reglas de negocio de la barbería.

## 📋 Contenidos

- **`informacion-negocio.md`** - Datos principales del negocio (ubicación, horarios, servicios, precios)
- **`servicios.md`** - Catálogo detallado de servicios ofrecidos
- **`horarios-disponibilidad.md`** - Horarios de apertura/cierre, descansos, personal disponible
- **`politicas.md`** - Políticas de cancelación, reprogramación, etc.
- **`preguntas-frecuentes.md`** - FAQs y respuestas predefinidas para el agente

## 🎯 Objetivo

Mantener la información del negocio centralizada y fácil de actualizar, sin mezclarla con la configuración técnica de n8n.
