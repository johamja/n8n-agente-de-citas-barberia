# 🛠️ Documentación Técnica

Esta carpeta contiene toda la documentación relacionada con la implementación técnica del agente de n8n para la barbería.

## 📋 Contenidos

- **`arquitectura.md`** - Descripción general de la arquitectura del sistema
- **`flujos-n8n.md`** - Documentación de los flujos/workflows de n8n
- **`integraciones.md`** - Detalles de integraciones (Google Calendar, WhatsApp, etc.)
- **`base-datos.md`** - Esquema de base de datos y estructura
- **`api-endpoints.md`** - Endpoints y webhooks utilizados
- **`variables-ambiente.md`** - Variables de entorno necesarias
- **`deployment.md`** - Instrucciones de deployment y configuración
- **`troubleshooting.md`** - Guía de problemas comunes y soluciones

## 🎯 Objetivo

Mantener la documentación técnica separada de la lógica de negocio, facilitando tanto mantenimiento como escalabilidad del sistema.

## 📝 Convenciones

- Los cambios técnicos se documentan en tiempo real
- Cada flujo de n8n debe tener descripción de entrada/salida
- Las integraciones incluyen ejemplos de payloads
- El versionado de documentación sigue las fechas de actualización
